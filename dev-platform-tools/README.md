# dev-platform

> 开发平台(工具)

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```

For a detailed explanation on how things work, check out the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).

## 测试环境

``` bash
# 统一登录
npm run test-login

# 数据标准管理：data-standard-manage
npm run test-data-standard-manage

# 规则中心：rule-center
npm run test-rule-center

# 元数据管理：meta-data-manage
npm run test-meta-data-manage

# 数据管理：data-manage
npm run test-data-manage

# 数据采集：data-collection
npm run test-data-collection

# 数据质量管理：data-quality-manage
npm run test-data-quality-manage

# ETL管理：etl-config
# npm run test-etl-config

# 数据开发(ETL)：data-dev-etl
npm run test-data-dev-etl

# 自助智能分析：self-service-analysis
npm run test-self-service-analysis

# 指标中心：indicator-center
npm run test-indicator-center

# 用户画像：user-portrait
npm run test-user-portrait
```
